
## 해치웠나

### 문제 설명

- 폭탄 ( 의 개수 만큼 정확히 레이저 ) 를 발사 하는 블록버스터<br>


### 자료 구조

- laser<br>
    - 타입 : 문자열
    - 저장 데이터 : 폭탄 ( , 레이저 ) 갯수 저장

- stack<br>
    - 타입 : 리스트
    - 저장 데이터 : 스택 공간처럼 사용할 저장소


### 풀이 과정

```txt
1. 입력을 받고 스택 리스트 선언

2. 폭탄이 나올 때만 스택에 저장하고, 레이저가 정확히 상쇄 하는지 확인

3. 스택에 폭탄이 남아있거나, 스택에 아무것도 없는에 레이저가 발사되면 출력

```

### 코드 구현
사용 언어 : 파이썬<br>

 
```python
# 풀이과정 1
def space():
    laser = input()
    stack = []

# 풀이과정 2
for i in laser :
        if i == '(':
            stack.append(i)
        else :
            if len(stack) == 0 :
                return "NO"
            stack.pop()

# 풀이과정 3
if len(stack) > 0:
        return "NO"
    return "YES"


```
