## 문제 설명

용사와 악당의 결투를 문자열로 받아 방어에 성공했는지 판별  <br>


## 자료 구조
- stack  <br>
타입 : str <br>
replace를 통해 저장되는 변수

## 풀이 과정
예시 :
```txt
1. stack 변수를 만들어줍니다.
2. replace 내장 함수를 이용하여 최대한 ()로 묶인 괄호를 제거합니다.
3. 조건을 넣어서 비교해줍니다.

```

## 코드 구현
```python

stack = input().replace("()", "").replace("()", "")                 

try:                                                                
    if stack[0] == ")" or stack[-1] == "(" or len(stack) % 2 == 1: 
        print("NO")
    else:
        print("YES")

except:
    print("YES")

```
